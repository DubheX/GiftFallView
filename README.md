# GiftFallView
---
红包雨落下的动画；

##使用
---

* GiftFallView


```

传入图片初始化：mGitfallview.init(bitmaps);

开始动画：mGitfallview.startFall();

结束动画： mGitfallview.stopFall();

```

* SimpleGiftFallView

简化版，红包落完后自动停止。

```

传入图片初始化：mGitfallview.init(bitmaps);

开始动画：mGitfallview.startFall();


```

## 注意
---

红包的下落速度与运算速度相关，在不同性能上的手机速度可能不同
